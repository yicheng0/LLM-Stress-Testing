# LLM Stress Testing

LLM Stress Testing is a performance benchmarking and stress testing toolkit for large language models.

It is designed to evaluate key model serving metrics under different workloads and concurrency levels, then generate structured performance reports.

## Key Metrics

- **TPM**: Tokens Per Minute
- **RPM**: Requests Per Minute
- **TPS**: Tokens Per Second
- **TTFT**: Time To First Token
- **Average Latency**: Average request completion time
- **P50 Latency**: Median response latency
- **P99 Latency**: Tail latency under high load
